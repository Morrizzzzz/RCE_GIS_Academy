# RCE GIS Academy
Op deze pagina wordt een aantal videos gegeven die fungeren als introductie op GIS en geodata en de relatie tot het te ontwikkelen Data Aggregatie platform in de context van het programma DNA-Nl.

## Video 1 Introductie op Geo data
Geeft een overzicht van de theorie rondom Geodata. Er wordt een onderscheidt gemaakt tussen discrete fenomemen en continious fenomenen in de realiteit en hoe die digitaal gerepresenteerd kunnen worden. Er wordt daarbij een odnerscheid gemaakt tussen Raster en Vector data. Daarnaast wordt een verschil gemaakt tussen 1) gemeten data, 2) geobserveerde data en 3) gemodelleerde data.

## Video 2 Introductie op GIS desktop applicaties - Deel 1 data exploratie 
Binnen het GIS desktop landschap zijn er een drietal "smaken" die in Nederland het meest gebruikt worden.
1. [ArcGIS pro](https://www.esri.nl/nl-nl/producten/arcgis-pro/overview) - Proprietary software - een van de marktleiders.
2. [MapINFO](https://www.precisely.com/product/precisely-mapinfo/mapinfo-pro/) - proprietary - minder gebruikt in Nederland.
3. [QGIS](https://qgis.org/) - Free and Open Source Software (advies is om de LTR te gebruiken)

In deze video worden basis begrippen van een desktop GIS toegelicht. Hoe zit deze software in elkaar, wat zijn data lagen en hoe kan je services van andere portalen inladen.
